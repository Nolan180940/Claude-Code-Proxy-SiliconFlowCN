Claude-Code-Proxy-SiliconFlowCN

Claude Code 对接国产大模型平台的轻量级API代理

轻量 · 兼容 · 国产化适配

仓库核心定位

一款打通 Claude Code 与国产大模型平台 SiliconFlow（硅基流动） 的轻量级API代理工具

核心目标

完成 Anthropic Messages API 与 OpenAI 兼容 API 之间的双向转换，让用户可以用 SiliconFlow 上的国产大模型驱动 Claude Code

核心工作原理

本地搭建中转服务（默认地址：http://127.0.0.1:8787），实现双向协议与数据流转换

请求转换

将 Claude Code 发出的 Anthropic 格式请求，转为 SiliconFlow 支持的 OpenAI 格式

响应转换

将 SiliconFlow 返回的 OpenAI 流式响应，转回 Anthropic 格式给 Claude Code

快速使用流程

1

前置准备

在 SiliconFlow 官网注册账号，获取 API Key（核心凭证）

2

环境配置

复制 .env.example 为 .env，填入 SILICONFLOW_API_KEY=你的密钥；或终端临时设置环境变量

3

启动使用

安装依赖：npm install

启动代理：node server.js

启动 Claude Code：指向本地代理地址 http://127.0.0.1:8787 即可
